# iacobus-images
Repositorio de imaxes utilizado para almacenar as miniaturas (thumbnails) e iconas que se mostran no catálogo Iacobus
Estas imaxes teñen prioridade sobre as procedentes de servizos externos como Google Books ou Syndetics, permitindo controlar de forma local a representación visual de determinados materiais, especialmente dos fondos históricos e Galicia. Estrutura do repositorio: As imaxes organízanse en carpetas segundo o tipo de material ou colección. Exemplo:incunables/ ; manuscritos/ ; comics/. Cada carpeta contén as imaxes correspondentes a ese tipo de material.
Convención de nomes: Para garantir a estabilidade das ligazóns recoméndase usar só minúsculas, evitar espazos e caracteres especiais, empregar guión baixo ou guión, utilizar formato .jpg sempre que sexa posible. Exemplos: comic_000045.jpg ; ms_000012.jpg.
Uso en Iacobus: As imaxes deste repositorio utilízanse como miniaturas para rexistros bibliográficos no catálogo. Nos rexistros MARC empregarase o seguinte campo local: 960 \ $a. No subcampo $a indicarase o identificador ou ruta da imaxe dentro deste repositorio. Exemplo nun rexistro MARC: 960 \ $a bhp/bhp_000123.jpg. A URL pública correspondente será: https://buscusc-lang.github.io/iacobus-images/bhp/bhp_000123.jpg
Mantemento: Este repositorio está destinado exclusivamente ao almacenamento de imaxes empregadas na visualización do catálogo. Calquera modificación na estrutura das carpetas ou nos nomes dos ficheiros pode afectar ás ligazóns utilizadas en Iacobus, polo que se recomenda manter unha estrutura estable e consistente. Recoméndase tamén manter unha nomenclatura coherente e documentada para facilitar a xestión das imaxes a longo prazo.
